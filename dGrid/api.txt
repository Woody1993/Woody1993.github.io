表格配置参数

1.表格父容器 box: 'selector'

2.数据来源 dataFrom: 'ajax|local'

3.获取表格数据
	a.当dataForm为ajax时

		1.ajax地址 url: ''

		2.ajax参数 data: {}

		3.每页条数 pageSize: 1

		4.ajax请求数据类型 dataType: 'text|json|xml'

		5.ajax请求数据格式化 dataFormatter: function

	b.当dataForm为local时

		1.表格数据 data: []

4.表格宽度 width: 1

5.表格高度 height: 1

6.索引列 index: true|function

7.选择模式 selectModel: 0|1|2 不能选|单选|多选

8.选择方式 callSelectModel: 0|1|2 通过点击复选框|通过点击行|两种都行

9.是否有全选 selectAll: true|false

10.列配置 colModel: []

11.是否分页 pageBar: false

12.行点击事件 rowOnClick: function



表格生成步骤

1.初始化一些配置

2.将列分组

3.生成表格框架

4.获取数据，填充表体行、汇总行、分页栏

5.统一行高

6.初始化表格大小

7.绑定事件



表格方法

1.刷新表格  update

	a.静态数据时传入新的数据，或使用老数据

	b.动态数据时传入页数，默认第一页

2.在表格最后插入n行  pushRows

3.在表格最前插入n行  unshiftRows

4.在表格指定位置插入n行 insertRows

5.删除指定n行 deleteRows

6.将指定位置的行移到指定位置 moveRowTo

7.刷新行的索引 1,2,3... updateIndex

8.选中指定的n行 selectRows

9.取消选中指定的n行 unselectRows

10.获取选中行的索引 getRowsIndex

11.获取选中行的数据 getRowsData

12.获取表格所有数据 getData

13.重置表格宽度 resizeWidth

14.重置表格高度 resizeHeight
